FlavorForge 🍽️
FlavorForge is a Django-based web application built for sellers to upload, manage, and showcase their food dishes.
It provides an easy-to-use platform for adding delicious recipes along with images.

✨ Features
Secure seller login and registration

Add new dishes with details and images

Edit or delete existing dishes

Organized and clean dashboard for management

🛠️ Tech Stack
Backend: Django
Database: SQLite
Frontend: HTML, CSS

🚀 How to Run Locally

Clone the repository:
git clone https://github.com/Vinamrakashyap/FlavorForge.git

Navigate to the project directory:
cd FlavorForge

Apply migrations:
python manage.py migrate

Start the server:
python manage.py runserver
Visit:
Open your browser and go to http://127.0.0.1:8000/ 🚀
